  1. 1

    เข้าไปยังเว็บไซต์ใดก็ได้

    ไปที่หน้าเว็บที่คุณต้องการใช้เครื่องมือความปลอดภัย

  2. 2

    กด Ctrl+Shift+F12

    ใช้ปุ่มลัดบนคีย์บอร์ด หรือคลิกที่ไอคอน extension ในแถบเครื่องมือ

  3. 3

    โหลดเครื่องมือความปลอดภัย

    ไปที่แท็บ Tools และคลิกที่เครื่องมือใดก็ได้เพื่อโหลดลงในหน้าเว็บ

  4. 4

    เริ่มแฮ็ก!

    ใช้แท็บ Console เพื่อรัน JavaScript หรือบันทึก code snippets
